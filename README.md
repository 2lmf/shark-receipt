# 🦈 ReceiptScanner: Vizualni Koncept

Ovdje možeš vidjeti kako sam zamislio sučelje za novi projekt.

## 🖼️ Dizajn Sučelja
Otvori sliku [mockup_orange.png](mockup_orange.png) u ovom folderu.

### Što vidiš na slici?
- **Premium Dark Mode**: Pitch Black (#010103) pozadina s vibrantnim narančastim (#E67E22) akcentima (klasični 2LMF stil).
- **Glassmorphism**: Kartice računa izgledaju kao zamućeno staklo s narančastim odsjajem.
- **Live Feed**: Računi se pojavljuju kao kartice, a AI odmah izdvaja ključne podatke označene narančastom bojom.
- **Shark Sync Gumb**: Glavna akcija koja pokreće obradu novih računa.

### Sljedeći koraci:
1. Potvrdi mi sviđa li ti se ovaj dizajn.
2. Definirat ćemo točno polja koja ti trebaju (OIB, PDV, Kategorija...).
3. Složit ćemo PWA koji se spaja na tvoj postojeći Google Sheet bez diranja stare logike.
